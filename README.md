\# Graphics Editor in C



A menu-driven 2D graphics editor built in C using characters (\* and \_).



\## Features

\- Draw Circle

\- Draw Rectangle

\- Draw Line

\- Draw Triangle

\- Add and Delete objects

\- Display the picture on a 2D canvas



\## How to Compile

gcc main.c -o graphics\_editor



\## How to Run

./graphics\_editor
