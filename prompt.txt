PROMPT 5:
How to create a GitHub repository and push code daily

RESPONSE 5:
1. Create repo on github.com
2. git init
3. git add .
4. git commit -m "first commit"
5. git remote add origin https://github.com/YOUR_USERNAME/graphics_editor.git
6. git branch -M main
7. git push -u origin main
Daily: git add . → git commit -m "message" → git push
